# my-new-repository

作为“学习”用的资料保存。

## 目标
- 记录学习笔记、示例与练习。
- 可重复运行的示例代码与答案参考。

## 目录
- docs/        - 学习笔记与教程
- examples/    - 示例代码
- exercises/   - 练习题与答案
- slides/      - 演示文稿

## 如何贡献
1. Fork 或 clone 本仓库。
2. 新建分支：`git checkout -b feat/your-topic`
3. 提交更改并发起 Pull Request。
4. 请在 PR 描述中写明你的更改内容和关联章节。

## 许可证
本仓库默认许可证：MIT（若你想用其它许可证，请在 LICENSE 中说明）。
